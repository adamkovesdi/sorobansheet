# Soroban worksheet generator in Python

Outputs a single page pdf with 25 excercises in addition and subtraction on soroban.

![soroban-serenity](soroban.jpg)

## Usage

```
python3 pdf.py
```
Outputs one file: output.pdf (A4, printable)
